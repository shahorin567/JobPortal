# University Job Portal

A comprehensive University Job Portal built with ReactJS and Firebase, connecting students and alumni with employers.

## Features

### 🔑 Authentication
* Email/password sign-up & login
* Secure session handling, password reset

### 🔍 Job Listings & Search
* Search & filter by title, company, type, location, etc.
* Detailed job descriptions
* Apply directly through portal

### 📄 Resume Upload & Profile
* Upload/manage resumes
* Resume builder
* Profile section with skills, education, experience

### 📧 Notifications
* Email/SMS/push for job alerts, status updates, deadlines

### 🧑‍💼 Employer Dashboard
* Post/manage job listings
* Resume search
* Interview scheduling
* Feedback for applicants

### 🌐 Networking & Engagement
* Alumni-student mentoring
* Discussion forums, groups
* Event/job fair listings

### 📊 Application Tracker
* View job application history & status
* Feedback from employers

### ⭐ Reviews & Ratings
* Candidates rate employers
* Employers rate applicants

## Getting Started

### Prerequisites
* Node.js (v14 or higher)
* npm or yarn

### Installation

   npm install
   npm start
